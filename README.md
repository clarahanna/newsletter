# AiMY Newsletter Component

This project is a React implementation of the AiMY newsletter design, showcasing various AI agents and their capabilities.

## Features

- Responsive newsletter layout
- Integration with Tailwind CSS for styling
- Display of AiMY's AI agents (QA, Recruitment, Knowledge, and Sales)
- Interactive sections with images and descriptive content

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

### Development

Run the development server:

```bash
npm start
```

The application will be available at [http://localhost:3000](http://localhost:3000).

### Building for Production

Create a production build:

```bash
npm run build
```

## Project Structure

- `/src` - Source code
  - `/components` - React components
  - `/assets` - Images and other static assets
  - `/styles` - CSS styles
